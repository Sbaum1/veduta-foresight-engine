To run Foresight X:
Place the veduta_foresight_x folder in veduta_project/ alongside forecast_engine/. The .streamlit/config.toml must be inside veduta_foresight_app/ for Streamlit to pick it up automatically.

# 1. Create the venv inside the Foresight X folder
cd C:\Dev\Repos\Veduta_Foresight_X
python -m venv .venv

# 2. Activate it
.venv\Scripts\Activate.ps1

# 3. Confirm you're in it — prompt should show (.venv)
python --version

cd C:\Dev\Repos\Veduta_Foresight_X
streamlit run app.py

pip install -r requirements.txt
